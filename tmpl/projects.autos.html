What is AutOS?

AutOS is a collection of scripts, that makes the post-installation process of Windows 10 + 11 or Arch Linux easier and quicker by automating it.

It is a fork of win10-regtweaks by nitrate92/HM.
GitHub repository

If you want to check out the project, go to it's GitHub repo, the link is there. Come on, do it. You know you want to.

Supported systems

 * ** ***

*Only runs on Fall Creators Update (v1709) and later

**This project is optimized for Windows 10, so beware of potential issues if ran on 11.

***This only supports Arch Linux and it's derivates! It will not work on any other distribution!

Languages used

PowerShell + Windows Registry (Windows)

Python (Linux)
