What is PubSite?

PubSite is a website that anyone can edit/make changes to. The goal was to see how much chaos would happen from creating what's basically an anarchy website.

GitHub repository

If you want to contribute code to PubSite, go the repository and knock yourself out.

URL

The live instance of PubSite can be found at pubsite.cf.

Supported systems

All. Your mileage may vary on some OS/browser configs.

Languages used

HTML, CSS, and JavaScript
